2D Platformer

- DirectX 
- C++
- HLSL
- No external libraries used
2D-Platformer built from the ground-up in DirectX.
Built without any external libraries, meaning that all physics, collision, animation, etc. was implemented by me.
The player, enemies and level is read in from a .png file.
The UV of the level is automatically assigned when reading in the level and is then used to read in specific parts of a tileset.
The purpose of showing this project in my portfolio is to show my proficiency in C++ and DirectX
