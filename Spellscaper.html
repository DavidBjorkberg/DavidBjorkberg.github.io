Spellscaper

Open World Multiplayer Sandbox

I worked on Spellscaper, a multiplayer open-world game where players can create gameplay actors in real-time by sending a prompt to an AI that models, codes, and spawns the actor.
This concept presented unique challenges, such as enabling Unreal Engine to compile new code at runtime and ensuring multiplayer stability and consistency across clients when new code and models are constantly being injected into the game.
I worked on the game from the initial prototype to the Beta version, focusing primarily on Multiplayer, UI, and Gameplay.
Responsibilities:
- Game stability
- Multiplayer consistency
- Gameplay features
- Support technical game designers
