Frame By Frame Debugger

Unreal Engine plugin that lets you record frames during gameplay and replay them for easy and visual debugging

I initially created this plugin to support the development of my RTS Movement plugin [Link]. I felt that the usual methods of debugging were not sufficient because I needed to see how forces interacted with each other and were modified both between frames and within a frame.

Plugin Functionality

- This plugin lets you record frames into a .json file, storing only the changes from the previous frame.
- The .json file contains any data that the user has decided to store each frame.
- A navmesh of the current map is also stored into an .obj file which the plugin then spawns when debugging.

When watching the replay you can:

- See the values of each variable
- Step through arrays
- Select an actor to see their stored variables
- Visualize vectors/shapes by checking the draw checkbox
