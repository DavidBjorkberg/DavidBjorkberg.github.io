2D Game Engine

I created a 2D game engine with the goal of building a simple 2D platformer from scratch.
The engine is built entirely without external libraries, meaning I implemented all core systems myself, including rendering, physics, collision detection, and animation.
It’s written in C++ with DirectX handling CPU-side rendering, and HLSL for GPU shaders.
The level and the placement of the player and enemies is read in from a .png file, acting as a level editor. The engine automatically maps UV coordinates when reading the level, using them to extract specific tiles from a tileset.
Notable Features:
- Keyboard input handling
- Spritesheet animation system
- Simple level editor
- Collision detection
- Raycasting
- UI components
- Texture management
