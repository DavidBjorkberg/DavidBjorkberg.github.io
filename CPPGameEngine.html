2D Game Engine

I created a 2D game engine with the goal of creating a simple 2D Platformer in it.
It is built without any external libraries, meaning that all rendering, physics, collision, animation, etc. was implemented by me.
The player, enemies and level is read in from a .png file. The UV of the level is automatically assigned when reading in the level and is then used to read in specific parts of a tileset.
Notable Features:
- Keyboard input
- Spritesheet Animation
- Simple level editor
- Collision
- Raycasts
- UI
- Textures
