2D Turnbased Strategy

A 2D turnbased strategy game created with Unity where the player places "mines" in the playable grid which triggers for different effects when an enemy walks over it. The mines and the player walk grid is rendered using custom shaders and the level / spawnpoints are created using Unitys tilemap system.
