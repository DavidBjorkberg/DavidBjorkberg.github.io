Unity Level Editor

The main feature of this project is the level editor.
The editor runs outside of play mode and has features such as gameobjects snapping to the floor/ceiling/walls, stretching to fit and custom unity editors to simplify the level creating process.
Many of these features are shown in the gifs below.
The game is a local multiplayer (1 keyboard + up to 3 controllers) arena brawler where, on each stage, the last player standing wins.
The purpose of showing this project in my portfolio is to display my in depth knowledge of Unity and C#.
