Unity DOTS

Reads in a mesh and spawns squares that fly to their assigned face, resulting in an entity that has the same shape as the mesh and that can break apart.
This was intended to be used for a larger project where the enemies would be created using this system. But at the moment dots is not developed enough to implement all the features the game would need, so i had to put it on hold.
The purpose of showing this project in my portfolio is to show that i have experience working in an ECS system
